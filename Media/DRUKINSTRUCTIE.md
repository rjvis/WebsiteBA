# Drukinstructie — BOB Autowas Wasclub poster

## Bestand
- **Drukklaar:** `BOB_Wasclub_poster_A0_drukklaar.pdf`
- **Bron (vector):** `bob_wasclub_poster_print.svg`

## Specificaties

| Eigenschap            | Waarde                                  |
|-----------------------|-----------------------------------------|
| Formaat eindproduct   | A0 staand · 841 × 1189 mm               |
| Formaat met afloop    | 847 × 1195 mm                           |
| Afloop                | 3 mm rondom                             |
| Kleurmodus            | CMYK                                    |
| Resolutie             | Vector (oneindig schaalbaar)            |
| Pagina's              | 1                                       |
| Oriëntatie            | Staand (portrait)                       |

## Belangrijke kleuren
- **BOB-rood:** #E30613 — bij voorkeur drukken als Pantone Red 032 C
  voor merkconsistentie (anders als CMYK 0 / 100 / 100 / 0)
- **Achtergrond:** #1a1a1a — rich black (CMYK 60 / 40 / 40 / 100) voor
  diepte en geen vlekkerige zwart-weergave
- **Wit (prijsblok losse wasbeurt):** zuiver wit, geen tint

## Lettertypen
Het drukklare PDF-bestand heeft de teksten als embedded fonts
(Arial Black + Arial). Mocht uw drukpipeline dit niet accepteren,
gebruik dan de SVG-bron en converteer eerst alle teksten naar paden
(bijv. via Inkscape → Path → Object to Path, of Illustrator →
Type → Create Outlines).

## Aandachtspunten
- De zwarte achtergrond loopt door tot in de afloopzone — geen
  witte randjes mogelijk bij het snijden.
- De rode boog rechtsboven is bewust subtiel (14% opacity) en mag niet
  worden versterkt.
- De Proteqt-labels op de prijsblokken steken bewust 25 mm boven de
  rand uit als 'sticker'-effect — graag zo behouden.

## Papier-aanbeveling
- 170-200 g/m² gestreken mat papier voor binnen-gebruik
- Eventueel mat lamineren voor extra duurzaamheid en mooi contrast
  tussen het diepe zwart en het rood
